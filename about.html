About the Latin America Map

The Latin America Map is a website based on Australia's NationalMap. The original NationalMap was repurposed to display information pertinent to any country in Central or South America, which included code changes and modification of the data sources.

Initially, as a proof of concept, the Latin America Map uses data from the Centro Nacional de Memoria Hist√≥rica, a Colombian public institution that investigates and monitors the evolution of armed conflict in the country.

To see what data is currently available on the map, refer to the Data Catalog in the map itself. Click the Data Catalog tab and expand the Data Set group names, if needed. Some of the datasets offer a time-lapse view of the information.

This application has been put together by:

  • Rafael Campos, Drexel University
  • Leizer Teran, Drexel University

How to use the map

Just go to http://latam-map.org/. Most operations are obvious, but if you need more help, you can refer to the original NationalMap Help and FAQs.

For further information email support@latam-map.org. Feel free to send us feedback and bug reports.

Contributing Data

We're eager to have more data added to this map. If you have suggestions of data, please contact us through support@latam-map.org.

Source Code

The NationalMap is Open Source software available as a Github project.
The core platform, TerriaJS, was initially developed by NICTA for the NationalMap, but has subsequently been used for other projects. It has been made available to assist with the development of other spatial data viewing applications.

The Latin America Map software contains changes specific to our application, and the source code can be found on GitHub.
