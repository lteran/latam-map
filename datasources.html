Data Sources

At this prototype stage, the Latin America Map uses data from the Centro Nacional de Memoria Histórica, a Colombian public institution that investigates and monitors the evolution of armed conflict in the country.

The datasets used are available for download at Bases de datos ¡Basta ya!.

To the information contained in those spreadsheets, we have added GPS coordinates obtained through geocoding using Google Maps and Bing Maps geocoding APIs.

The resulting CSV files can be downloaded from the Information pane on the Data Catalog tab of the Explorer menu bar in the main application.

Next Steps

We intend to augment the currently presented data with information from other sources, including socio-economical data that might contribute to the interpretation of the existing datasets.

Contributing data

We're eager to have more data added to this map. If you have suggestions of data, please contact us through support@latam-map.org.
